Szerencsejáték - ötös lottó

Program egy adatbázisból felveszi a szelvényeket 
	- 1 szelvényen 5 db 1-90 intervallumban lévő egész szám szerepel

Random szám generátorral kisorsol 5 db, a fenti intervallumba eső számot. Ezek a nyerőszámok.

Ezután a program megnézi, hogy hány olyan szelvény van amin volt találat.
Ezeket, egy excelbe kiírja.

A 0 találatos szelvényeket a program törli.
